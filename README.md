### 🚀 Featured Industrial Projects

- 🏗️ **CRM Joglo (AI-Driven Business Suite)**
  - **RAG Architecture**: Full implementation of **Retrieval-Augmented Generation** using **Google Gemini Pro** for embeddings/generation and **Qdrant Vector Database** for efficient semantic search.
  - **Intelligent Knowledge Base**: Automated synchronization of business documents into vector stores to power context-aware AI sales assistants.
  - **Modern Stack**: Built with **Node.js**, **Prisma**, **React 18**, and high-end animations via **Framer Motion** & **GSAP**.

  - Complex business logic implementation and module customization using **Python** and the **Odoo 17** framework.
